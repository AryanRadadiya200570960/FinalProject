API Description and Source

In this project we have used coinmarketcap API. Which is a free opensource api for crypto data. It gives real time data for crypto prices, blockchain resources, records or NFTs and many more. The CoinMarketCap API is a suite of high-performance RESTful JSON endpoints that are specifically designed to meet the mission-critical demands of application developers, data scientists, and enterprise business platforms.


Source: https://pro.coinmarketcap.com/api/v1


Cloud Storage

We are using MongoDB cloud server to store our data from the API and then fetch data from the cloud server and visualize it. The script will run every 24hr and we will get live real time data on our website in form of charts.


Data Cleaning

Then we will fetch data from the cloud server and clean it, so that we can proper visualize it and give meaningfull insites from the data. The data from the cloud server will be in JSON formate and we will convert it into DataFrame. After that we will remove the unwanted rows and just use those rows on which we will be visualizing.
